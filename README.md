# Eventra  

Eventra is a web application designed to keep students connected with all the events happening on campus. It centralizes event details, notifies students about upcoming activities, and allows easy participation and enrollment.  

## 🚀 Features  
- Centralized dashboard of all campus events  
- Event notifications for students  
- Easy participation and enrollment process  
- Admin panel to manage and update events  

## 🛠️ Tech Stack  
- Frontend: HTML, CSS, JS
- Backend: PHP
- Database: SQL
